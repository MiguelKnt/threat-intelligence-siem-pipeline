<h1>🚨 Threat Intelligence Pipeline & SIEM Detection Framework</h1>

<h2>📌 Overview</h2>
<p>
This project demonstrates the design and implementation of an end-to-end threat intelligence pipeline leveraging Python automation and Splunk for detection, analysis, and incident response.
</p>

<p>
The system integrates automated threat intelligence collection, data normalization, SIEM ingestion, correlation-based detection, alerting, visualization, enrichment, and incident response workflows.
</p>

<hr>

<h2>⚙️ Technologies Used</h2>
<ul>
  <li>Python (automation & data processing)</li>
  <li>Splunk (SIEM, detection, alerting, dashboards)</li>
  <li>Ubuntu Server (lab environment)</li>
  <li>VirtualBox (virtualization)</li>
  <li>SSH (secure file transfer)</li>
</ul>

<hr>

<h2>🔁 Pipeline Architecture</h2>
<p>
Threat Intelligence → Collection → Normalization → SIEM → Detection → Alerting → Visualization → Enrichment → Incident Response
</p>
